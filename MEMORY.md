# MEMORY — garcia

## Rule Zero

**ПЕРЕД відповіддю на будь-яке питання про стан проекту garcia — першим ділом запитати у Саші вміст HOT.md + WARM.md.** Не відповідати з пам'яті, не вгадувати.

Команда для Саші: `cat /home/sashok/.openclaw/workspace/garcia/HOT.md /home/sashok/.openclaw/workspace/garcia/WARM.md`

## Триярусна пам'ять

- **HOT.md** — ~60 рядків, переписується щосесії. Поточний фокус, останнє що робили, наступний крок, блокери.
- **WARM.md** — ~400 рядків, інкрементальні оновлення. Архітектура, рішення, стан компонентів, відкриті питання.
- **COLD.md** — append-only історія. Завершені фази, міграції, рефакторинги в хронологічному порядку.
